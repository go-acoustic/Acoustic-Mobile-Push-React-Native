## Install
Add 'react-native-acoustic-mobile-push-inapp' to the project.

```shell yarn
yarn add 'react-native-acoustic-mobile-push-inapp'
```

```shell npm
npm i 'react-native-acoustic-mobile-push-inapp'
```
